# Support

- **Bugs / feature requests:** open a GitHub issue in this repository.
- **Setup help:** run **Voice Agent: Health Check**, then **Voice Agent: Setup Whisper**.
- **Billing / Pro license:** use **Voice Agent: Manage Subscription / Checkout** or **Activate Pro License**.
- **Privacy questions:** see [PRIVACY.md](PRIVACY.md).

Pilot / soft-launch feedback is welcome: include OS, VS Code/Cursor version, LLM provider, and the health-check report (no secrets).
